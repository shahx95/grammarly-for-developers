Suggestion categories (Passive voice suggestions turned "on")

The Grammarly Text Editor Plugin on this page is set to show suggestions for the "passiveVoice" writing suggestion category. This category, which is muted by default, adds writing suggestions regarding the use of the passive voice.

Try the Grammarly Text Editor Plugin in the text area below!

In this example, writing suggestions for avoiding the passive voice in favor of the active voice are turned "on".

The example code was written by Andrew.

Configuring and muting suggestion categories is available only on the Grammarly Text Editor SDK Plus plan.
