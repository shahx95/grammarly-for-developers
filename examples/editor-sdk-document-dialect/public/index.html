Document Dialect (Auto Browser)

By default, Grammarly sets an appropriate dialect based on your user's browser settings ("auto-browser").

Alternatively, you can configure Grammarly to set an appropriate dialect based on your user's written text ("auto-text").

Try the Grammarly Editor Plugin in the text area below!

I like the color green. But not the colour blue so much.

To view the detected/configured dialect, click the Grammarly button, and then review the dialect value in the "I write in" dropdown list. Note that your users can always override this value to match their own dialect preferences.
